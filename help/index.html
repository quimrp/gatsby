C) Installation

Node.js and NPM
You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.

Install NPM Packages
Open deploy folder with Visual Studio Code (or any other you like). And follow step below:

NPM
You must have to run npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies. If it gives error/warning you can run it as npm install --force

Development Server
Run npm run dev or yarn dev for a dev server.
Navigate to http://localhost:8000. The App will automatically reload if you change any of the source files.

Build For Production And Launch Server
Run npm run build and then npm run start

Read deployment document on Gatsby.

E) Color Schema

Naxos comes with 8 predefined colors blue, green, red, turquoise, purple, orange, yellow and grey.

You can change the title and color of each page by the following variable:

<Layout 
    pageTitle="Naxos - React Gatsby App Landing Page Template" 
    colorSchema="../colors/blue.css" 
>

F) Background Image - Video

Single Image

Open components/Intro/SingleImage.js in a text editor. You'll see the following code:

<BackgroundSlider
    duration={100000}
    transition={0.75}
    className="image-bg"
    images={['/images/banner/single-image.jpg']}

Image Slideshow

Open components/Intro/SlideShow.js in a text editor. You'll see the following code:

<BackgroundSlider
    duration={100000}
    transition={0.75}
    className="image-bg"
    images={[
        '/images/banner/slideshow-1.jpg',
        '/images/banner/slideshow-2.jpg',
        '/images/banner/slideshow-3.jpg'

Youtube Background

Open components/Intro/VideoBackground.js in a text editor. You'll see the following code:

<YouTube 
    videoId='mqEeWiRwv0k' 
    className="video-bg" 
    containerClassName="video-bg-wrapper" 
    opts = {opts} 
/gt;

You can the video by replacing mqEeWiRwv0k with your own Youtube video ID.

G) Mailchimp

Mailchimp is intergrated to Naxos.

Mailchimp (subscription) in the file changes main.js.

In url parameter you have to put your signup form URL. Follow these steps to get this URL address:

  1. Login to Mailchimp and go to Audience
  2. Open your chosen list and click on Signup forms
  3. Click on General forms
  4. Find Signup form URL, copy the URL
  5. Open in your browser this page: www.unshorten.it, paste Signup form URL and press unshorten button
  6. You'll resolve URL similar to this one: 
https://USERNAME.SERVER.list-manage.com/subscribe/post-json?u=USER_ID&id=AUDIENCE_ID

Replace UPPERCASE variables at the subscription url which is located in components/Sections/Subscribe.js:

https://USERNAME.SERVER.list-manage.com/subscribe/post-json?u=USER_ID&id=AUDIENCE_ID

H) Contact Form

Contact form is integrated with Formspree - https://formspree.io/.

Create an account at Formspree.

Open Integration tab. You'll see the form url like below:

https://formspree.io/f/xlezjyrv

Last division is the form ID.

Search & replace FORMSPREE_FORM_ID on the following files:

  • components/Sections/Contact.js
  • components/Contact/ContactForm.js

J) RTL Languages

Firstly, add following HTML tag to the beginning of the components/Sections/Layout.js as the main container:

<bdo dir="rtl" lang="COUNTRY CODE">
    <div>
        ...
    </div>
</bdo>

Then, import Bootstrap CSS file with RTL version:

import 'bootstrap/dist/css/bootstrap.rtl.min.css';

After that, import rtl.css:

import '~assets/css/rtl.css';

Finally, edit components/Sections/Testimonials.js and replace all rtl: false:

rtl: true
